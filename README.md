# 🔮 命运工坊 - 综合占卜与命理应用

一个精美的综合占卜与命理 Web 应用，集成了多种传统和现代占卜方式，包括塔罗牌、八字算命、紫微斗数、姓名测试、生肖配对、奇门遁甲、风水罗盘等功能。

🌐 **在线访问**: [https://www.fateatelier.cloud/](https://www.fateatelier.cloud/)

## ✨ 功能特性

### 🔮 塔罗占卜
- 🎴 **完整牌组**：包含78张塔罗牌的完整信息（22张大阿卡纳 + 56张小阿卡纳）
- 🔮 **单张抽牌**：随机抽取一张塔罗牌
- 🌟 **三牌占卜**：一次性抽取三张牌进行占卜（过去、现在、未来）
- 🔄 **正逆位切换**：每张牌都可以查看正位和逆位的含义
- 📖 **详细解释**：每张牌都包含详细的牌意和描述
- 📚 **历史记录**：保存占卜历史，支持导出和分享
- ⭐ **收藏功能**：收藏喜欢的牌和占卜结果
- 📊 **统计分析**：查看占卜统计信息

## 🚀 快速开始

### 环境要求

- Node.js >= 16.0.0
- npm >= 7.0.0

### 安装依赖

```bash
npm install
```

### 启动开发服务器

```bash
npm run dev
```

应用将在 `http://localhost:5173` 启动

### 构建生产版本

```bash
npm run build
```

### 预览生产构建

```bash
npm run preview
```

### 代码检查

```bash
npm run lint
```

## 🎯 使用说明

### 塔罗占卜
1. **抽取单张牌**：点击"抽取一张牌"按钮，随机抽取一张塔罗牌
2. **三牌占卜**：点击"三牌占卜"按钮，一次性抽取三张牌（过去、现在、未来）
3. **查看牌意**：抽取后可以查看牌的正位和逆位含义
4. **翻转卡片**：点击卡片或"转为逆位/正位"按钮来切换正逆位
5. **选择已抽牌**：在已抽取的牌列表中点击任意一张牌来查看详情
6. **收藏功能**：点击收藏按钮保存喜欢的牌
7. **历史记录**：查看所有占卜历史，支持导出和分享
8. **重新开始**：点击"重新开始"按钮清空所有已抽取的牌

### 姓名测试
1. 输入姓氏和名字
2. 系统自动计算五格数理（天格、人格、地格、外格、总格）
3. 查看五行分析和三才配置
4. 获得姓名评分和详细解读
5. 了解姓名吉凶和建议

### 紫微斗数
1. 选择出生日期（支持农历/阳历）
2. 输入出生时辰
3. 系统自动生成完整命盘
4. 查看12宫位的主星、辅星分布
5. 了解四化（化禄、化权、化科、化忌）
6. 查看各宫位的详细分析

### 生肖配对
1. 选择第一个生肖
2. 选择第二个生肖
3. 查看配对指数（0-100分）
4. 了解配对关系（六合、三合、六冲、六害、三刑）
5. 查看兼容性等级和详细分析

### 每日幸运色
1. 选择查询日期
2. 可选择输入个人信息（出生日期、生肖、星座）获得个性化推荐
3. 支持农历/阳历转换
4. 查看幸运色详细解读和使用建议
5. 复制颜色代码（HEX、RGB）

### 数字能量
1. 输入数字进行分析
2. 查看数字组合的能量解读
3. 了解每个数字的详细含义
4. 获得个性化数字建议

### 奇门遁甲
1. 选择时间和方位
2. 查看九宫格盘面分析
3. 了解各宫位的八门、九星、八神
4. 获取吉凶方位建议
5. 查看时间选择建议

### 其他功能
- 通过3D旋转轮播选择不同功能
- 每个功能都有详细的说明和指导
- 支持移动端和桌面端访问
- 响应式设计，适配各种屏幕尺寸

## 🎨 牌组说明

### 大阿卡纳 (Major Arcana)
22张牌，代表人生的重要阶段和转折点，包括：
- 愚者、魔术师、女祭司、皇后、皇帝、教皇
- 恋人、战车、力量、隐者、命运之轮、正义
- 倒吊人、死神、节制、恶魔、塔、星星
- 月亮、太阳、审判、世界

### 小阿卡纳 (Minor Arcana)
56张牌，分为四个花色：
- **权杖 (Wands)**：代表行动、创造力和热情
- **圣杯 (Cups)**：代表情感、直觉和关系
- **宝剑 (Swords)**：代表思想、沟通和冲突
- **星币 (Pentacles)**：代表物质、工作和资源

每个花色包含14张牌：Ace到10，以及侍从、骑士、皇后、国王。
