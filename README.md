# 🔮 命运工坊 - 塔罗牌占卜应用

一个精美的塔罗牌占卜Web应用，包含完整的78张塔罗牌（22张大阿卡纳 + 56张小阿卡纳），支持单张抽牌、三牌占卜等功能。

## ✨ 功能特性

- 🎴 **完整牌组**：包含78张塔罗牌的完整信息
- 🔮 **单张抽牌**：随机抽取一张塔罗牌
- 🌟 **三牌占卜**：一次性抽取三张牌进行占卜
- 🔄 **正逆位切换**：每张牌都可以查看正位和逆位的含义
- 📖 **详细解释**：每张牌都包含详细的牌意和描述
- 🎨 **精美UI**：现代化的界面设计，支持响应式布局
- 💫 **动画效果**：流畅的卡片翻转和交互动画

## 🚀 快速开始

### 安装依赖

```bash
npm install
```

### 启动开发服务器

```bash
npm run dev
```

应用将在 `http://localhost:5173` 启动

### 构建生产版本

```bash
npm run build
```

### 预览生产构建

```bash
npm run preview
```

## 🎯 使用说明

1. **抽取单张牌**：点击"抽取一张牌"按钮，随机抽取一张塔罗牌
2. **三牌占卜**：点击"三牌占卜"按钮，一次性抽取三张牌
3. **查看牌意**：抽取后可以查看牌的正位和逆位含义
4. **翻转卡片**：点击卡片或"转为逆位/正位"按钮来切换正逆位
5. **选择已抽牌**：在已抽取的牌列表中点击任意一张牌来查看详情
6. **重新开始**：点击"重新开始"按钮清空所有已抽取的牌

## 🎨 牌组说明

### 大阿卡纳 (Major Arcana)
22张牌，代表人生的重要阶段和转折点，包括：
- 愚者、魔术师、女祭司、皇后、皇帝、教皇
- 恋人、战车、力量、隐者、命运之轮、正义
- 倒吊人、死神、节制、恶魔、塔、星星
- 月亮、太阳、审判、世界

### 小阿卡纳 (Minor Arcana)
56张牌，分为四个花色：
- **权杖 (Wands)**：代表行动、创造力和热情
- **圣杯 (Cups)**：代表情感、直觉和关系
- **宝剑 (Swords)**：代表思想、沟通和冲突
- **星币 (Pentacles)**：代表物质、工作和资源

每个花色包含14张牌：Ace到10，以及侍从、骑士、皇后、国王。

## 📝 注意事项

- 本应用仅供娱乐参考，塔罗牌占卜结果不应作为重要决策的唯一依据
- 每张牌的正逆位是随机决定的
- 抽取的牌不会重复，直到所有78张牌都被抽取完毕
